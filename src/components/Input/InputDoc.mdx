## Props

<div className="docsTable docsProps">

| Prop     | Type                                   | Default | Description                                 |
| -------- | -------------------------------------- | ------- | ------------------------------------------- |
| children | React.ReactNode                        | —       | Label text for input                        |
| id       | string                                 | —       | Used for input id and label htmlFor linking |
| name     | string                                 | —       | Name used in form submission                |
| type     | "text" \| "number" \| "email" \| "tel" | "text"  | Input field type                            |
| required | boolean                                | false   | Marks input as required                     |
| size     | number                                 | 30      | Width of input in characters                |

</div>
