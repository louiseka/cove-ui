## Props

<div className="docs">

| Prop     | Type                     | Default   | Description         |
| -------- | ------------------------ | --------- | ------------------- |
| children | React.ReactNode          | —         | Button content      |
| type     | "button" \| "submit"     | "button"  | HTML button type    |
| variant  | "primary" \| "secondary" | "primary" | Visual style        |
| disabled | boolean                  | false     | Disables the button |
