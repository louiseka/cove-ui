# Cove UI – React + TypeScript Design System

## Overview
This project is currently a work in progress.

Cove UI is a lightweight design system built with React and TypeScript. It focuses on reusable, type-safe UI components with consistent styling and simple internal documentation.

The project is designed to demonstrate frontend engineering skills including component architecture, TypeScript usage, styling patterns, and basic testing.

---

## Features to implement

- Reusable component architecture
- Type-safe props with TypeScript
- Scoped styling using CSS Modules
- Component-level documentation pages
- Unit testing with Vitest and Testing Library
- Sidebar-based navigation for component exploration

---

## Tech Stack
- React
- TypeScript
- CSS Modules
- Vite
- Vitest
